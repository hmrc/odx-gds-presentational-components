## Package notes - Dev

A react implementation of Government Design Standards (GDS) defined frontend components.

# How to consume
NPM install the package
```npm install gds-react-component-library```

Import components as named imports from the top level of the package.

# Documentation
For details on implementation, view the storybook documentation.
hosted at: tbc
Or generate the documentation locally. This can be done by checking out the repo, and, after installing packages
running
```npm run storybook```
Which will build and locally host the storybook documentation for the version which you have checked out.

# How to contribute
If you notice any issues, missing functionality or have any feedback, please raise an issue

Alternatively, you can propose changes by raising a Merge Request.
