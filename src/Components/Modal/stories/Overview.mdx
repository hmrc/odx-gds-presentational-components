## Details

The Modal component takes some modal content, and handles the modal display, rendering an overlay over the page, and the modal component, containing passed children.

It also accepts, and expects an `ariaTitleId` prop, which should match the id attribute of the title (h1, h2, h3 etc) and any other relevant label content provided in the children to make the modal accessible.
(N.B. This could be a list of ids, useful if the content contains some hidden, supporting text to announce alongside the h1 title)

<Canvas of={ModalStories.Default} height="800px" />

The component also handles some other expected default behaviour:

- Focus: The component will automatically move focus to the modal on rendering, and trap focus within the modal, stopping focus from leaving the elements within the modal.

- Escape key handling: The component will call the passed 'onClose' prop on escape key down events, while the modal is open. This is expected to be provided to support accesibility.
