# Radio Button Group
[Gov Design System Documentation](https://design-system.service.gov.uk/components/radios/)

The Radio button group component is an implementation of the Gov Design System Radio button group.
This component is expected to be used for questions for which a user can only select one option from a list.

<ArgsTable of={RadioButtonGroup} />

## Details

The Radio button group **requires** an id property, and a [list of options](/docs/gds-components-radiobutton-radiobutton--overview) and optional text dividers
The id property will be used as the name for options

An option is made up of required ```value``` and ```label``` properties, with optional ```conditionalContent``` ```hintText``` properties.
For example ```{value: 'yes', label: 'Yes', hintText: 'Yes hint text'}```
  The name of each option will be the id of the group, and the id of each option will be generated from the id of the group, and it's order

A divider is defined as an object with a divider property e.g. ```{divider: 'Or'}```

### onChange handling
By default, the radio buttons in the group will use a changeHandler that simply checks a radio button when checked.
For custom change handling, pass an ```onChange``` function to the group, which will be applied as onChange property to All radio buttons.
To define which radiobutton is currently selected, pass the value of the selected radio button as the ```value``` property of the group
