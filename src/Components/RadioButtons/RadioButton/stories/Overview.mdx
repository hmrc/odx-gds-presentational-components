# Radio Button
[Gov Design System Documentation](https://design-system.service.gov.uk/components/radios/)

<ArgsTable of={RadioButton} />

## Details

This is an implementation of a single radio button input, inline with Gov Design Systen
In general, it is not expected that this component will be used in isolation and instead the [Radio Button Group](/docs/gds-components-radiobutton-radiobuttongroup--overview)
will be used for radio button implementations (as the group handles the fieldset wrapper and other configuration options)

An individual Radio Button requires an id, name, value and label property
<Canvas of={RadioButtonStories.Default}/>

The checked status of the radio button is managed with the defaultChecked property.

A radio button can accept an optional hintText property which is displayed after it's label
<Canvas of={RadioButtonStories.WithHintText}/>

A radio button can accept an optional conditionally rendered element, which will be presented after the radio button, if it is checked
<Canvas of={RadioButtonStories.WithConditionalContent}/>
