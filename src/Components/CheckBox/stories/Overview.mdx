# Check Box
[Gov Design System Documentation](https://design-system.service.gov.uk/components/checkboxes/)

<ArgsTable of={CheckBox} />

## Details

This is an implementation of a single CheckBox input, inline with Gov Design Systen

An individual CheckBox requires an ```id```, ```name```, ```value``` and label property
<Canvas of={CheckBoxStories.Default}/>

The checked status of the CheckBox button is managed with the ```defaultChecked``` property.

A checkBox button can accept an optional ```hintText``` property which is displayed after it's label
<Canvas of={CheckBoxStories.WithHintText}/>

A checkBox button can accept an optional conditionally rendered element, which will be presented after the CheckBox, if it is checked
<Canvas of={CheckBoxStories.WithConditionalContent}/>
