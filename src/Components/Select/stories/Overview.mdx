# Select
[Gov Design System Documentation](https://design-system.service.gov.uk/components/select/)

The select component allows users to choose an option from a list.

This component should only be used as a last resort due to accessibility issues. Some users may struggle to close the select
or may try to type into the select component. Alternative approaches such as asking questions, text input or radio buttons are
suggested.
