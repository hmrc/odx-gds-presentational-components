# GOV.UK Footer
[Gov Design System Documentation](https://design-system.service.gov.uk/components/footer/)

The Footer component provides copyright, licensing and other information about your service. Use the footer at the bottom of every page in your service.

## Props

<ArgsTable of={GOVUKFooter} />

**Note:** as of June 25th 2025, this component will be affected by a GOV.UK brand refresh. The consuming application will handle applying the
``govuk-template--rebranded`` class for certain visual updates. The user can toggle ``brandRefresh`` to true or false to denote whether or not
the additional small crown will be displayed within the footer. This is to allow for differing time scales as applications gradually adopt the
brand refresh.

## Details

As a default, the component should display a copyright notice and the coat of arms. The type of licence should also be displayed. Use the Open Government
Licence unless the National Archives have permitted the use of another type of licence. This default displays no links.
<Canvas of={GOVUKFooterStories.Default}/>

The component can also be displayed with links.
<Canvas of={GOVUKFooterStories.WithLinks}/>

Secondary navigation can be handled within the footer, but this should only be used when creating a GOV.UK service, to let a user navigate away from this
service to somewhere else on the GOV.UK website.
<Canvas of={GOVUKFooterStories.WithSecondaryNavigation} />
