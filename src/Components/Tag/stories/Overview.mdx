## Details

### Usage

A default tag implementation requires the providing of:
- ```text``` property - Defining the text shown in on the tag
- ```colour``` property - Defining the colour for the tag

The default style of the tag is:
<Canvas of={TagStories.Default}/>

### Colours
For different colours of the tag, use the
```colours``` property, which expects a value from the TagColour enumerator
